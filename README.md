# E-Commerce Web Application

A responsive E-Commerce Web Application built using HTML, CSS, and JavaScript.

## 📌 Project Overview

This project is a simple online shopping web application where users can:

- Browse products
- Add products to cart
- Remove products from cart
- View total cart amount
- Login using a simple login form

The application provides a modern shopping interface with responsive design for desktop and mobile devices.
